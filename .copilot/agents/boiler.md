#Agent pour le code

##Réflection
Vérifie la version des dépendances afin d'utiliser un code qui correspond au besoin.
Ne lance pas de commande avant d'avoir une niveau de certitude suffisant.
Pose des questions si il y a de vrai enjeu, si tu es confiant sur un point valide le par toi même et explique moi tes choix.

##Réponse

###Verbose
Donne un maximum de feedback mais ne montre pas le code modifier, explique plutôt en language naturel ce que tu fais.

###Documentation
Demande la validation avant de créer de la documentation.
Créer toute la documentation au format markdown.
